# 基于SpringBoot+Vue的火车购票系统

![Spring Boot](https://img.shields.io/badge/-Spring%20Boot-6DB33F?style=flat-square&logo=spring-boot&logoColor=white) ![Vue3](https://img.shields.io/badge/-Vue3-grey?style=flat-square) ![ElementPlus](https://img.shields.io/badge/-ElementPlus-grey?style=flat-square) ![MySQL](https://img.shields.io/badge/-MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white) 

## 项目介绍

本系统是一个功能完善的火车票在线预订平台，采用前后端分离架构开发。用户可以查询车次信息、在线购票、管理订单、进行实名认证等操作；管理员可以通过后台管理车次、用户、订单及审核实名认证申请。系统界面美观，操作流畅，数据展示直观。

### 功能模块

- **车次信息展示**：首页实时展示今日列车信息，包括车次、出发站、到达站、时间、票价等
- **车票查询与购买**：支持按出发地、目的地、日期查询车票并在线购买
- **订单管理**：用户可查看所有订单，支持改签、退票等操作
- **个人中心**：用户个人信息管理、头像上传、密码修改
- **实名认证**：用户提交实名信息进行身份验证
- **管理后台控制台**：统计用户数、订单数、车次数等关键数据
- **车次管理**：管理员对车次信息进行增删改查
- **用户管理**：管理员查看和管理系统用户，设置角色权限
- **订单管理**：管理员查看所有订单，修改订单状态
- **认证审核**：管理员审核用户的实名认证申请

### 技术架构

- **后端**：Spring Boot 2.0.7 + MyBatis 1.3.2 + Druid 连接池
- **前端**：Vue 3.5 + Element Plus 2.4 + Vite 7 + Pinia + Vue Router 4
- **数据库**：MySQL 8.0
